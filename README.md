# Sample for Protractor.

> CircleCI + Protractor + WebdriverManager

## Usage

> local env.

```
# Install dependency
yarn install --save

# Download all drivers
yarn run webdriver:update

# Run Server
yarn run webdriver:start

# Run Test
yarn run test
```

## CircleCI

check `./.circleci/config.yml`
